1.zzrouter://productdetail
2.zzrouter://orderlist?orderid=xxx
3.zzrouter://category
4.zzrouter://homepage

## Installation

ZZRouter is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "ZZRouter"
```
